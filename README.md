## 说明
常见爬虫的封装
包括  selenium请求到网页内容  -->  [{字典:xpath},.....] 或者 url时[{网址:网址,字段名:xpath表达式,},.....])
      get请求到网页内容 --> [{字典:xpath},.....] 或者 url时[{网址:网址,字段名:xpath表达式,},.....])
      get 或者 post请求进行接口数据请求  --> [{字典:xpath},.....] 
     
     一个任务一个线程，每个线程/任务 使用线程池实现
     线程 + 线程池
     一个线程id对应一个任务id 
     
    如果返回值要求为网址那么带上该网址重新递归调用爬虫方法
